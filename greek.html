Δομές δεδομένων

Σύμφωνα με τους όρους ενός υπολογιστή, μια δομή δεδομένων είναι ένας συγκεκριμένος τρόπος αποθήκευσης και οργάνωσης δεδομένων στην μνήμη ενός υπολογιστή, ώστε αυτά τα δεδομένα να μπορούν να χρησιμοποιηθούν αποτελεσματικά αργότερα.Τα δεδομένα μπορούν να τακτοποιηθούν με πολλούς διαφορετικούς τρόπους, όπως το λογικό ή μαθηματικό μοντέλο.

    Το είδος ενός συγκεκριμένου μοντέλου δεδομένων εξαρτάται από τους δύο παράγοντες.

  • 1. Πρέπει να φορτωθεί αρκετά στη δομή ώστε να αντικατοπτρίζει τις πραγματικές σχέσεις των δεδομένων με το αντικείμενο του πραγματικού κόσμου.
  • 2. Ο σχηματισμός θα πρέπει να είναι αρκετά απλός, ώστε ο καθένας να μπορεί να επεξεργάζεται αποτελεσματικά τα δεδομένα κάθε φορά που είναι απαραίτητο.

Οι δομές δεδομένων χωρίζονται σε δύο τύπους:

  • a. Γραμμικές δομές δεδομένων
  • b. Μη γραμμικές δομών δεδομένων

Δομές δεδομένων πίνακα

Ένας πίνακας είναι ένα σύνολο από στοιχεία που αποθηκεύονται σε γειτονικές θέσεις μνήμης. Η ιδέα που επικρατεί είναι να αποθηκεύονται πολλά αντικείμενα του ίδιου τύπου. Με αυτό τον τρόπο μπορεί να υπολογιστεί η θέση κάθε στοιχείου αφού γίνεται μετατόπιση σε μια τιμή βάσης, δηλαδή στην θέση μνήμης του πρώτου στοιχείου πίνακα. Ο πίνακας μπορεί να περιέχει ένα καθορισμένο αριθμό αντικειμένων και αυτά τα αντικείμενα θα πρέπει να είναι του ίδιου τύπου. Οι περισσσότερες δομές δεδομένων υλοποποιούνται με την μορφή πίνακα για την εφαρμογή των αλγόριθμων τους.

Σημαντικοί όροι για να γίνει κατανοητός ο όρος του πίνακα είναι οι εξής:

  • 1. Στοιχεί:Τα περιεχόμενα του πίνακα ονομάζονται στοιχεία.
  • 2. Θέση μνήμης: Κάθε στοιχείο ενός πίνακα ανήκει σε μια θέση η οποία ονομάζεται θέση μνήμης και χρησιμοποιείται για την αναγνώριση του στοιχείου.

Linked List Data Structures

Μια συνδεδεμένη λίστα είναι μια ακολουθία δομών δεδομένων, όπου τα στοιχεία που περιέχει συνδέονται μεταξύ τους. Η ακολουθία των συνδέσμων που δημιουργείται αναπτύσσεται συνεχώς.Αποτελεί ένα σύνολο στοιχείων όπου κάθε στοιχείο αποτελεί τμήμα ενός κόμβου ο οποίος περιέχει επίσης ένα σύνδεσμο προς κάποιον κόμβο.Η συνδεδεμένη λίστα είναι η δεύτερη πιο συχνά χρησιμοποιούμενη δομή δεδομένων μετά τον πίνακα.

Σημαντικοί όροι για να γίνει κατανοητή η έννοια της συνδεδεμένης λίστας:

  • 1. Σύνδεσμος: Κάθε σύνδεσμος μιας συνδεδεμένης λίστας μπορεί να αποθηκεύσει δεδομένα.
  • 2. Επόμενο: Το στοιχείο αυτό είναι ένας δείκτης σε επόμενο στοιχείο.
  • 3. Συνδεδεμένη λίστα: Μια συνδεδεμένη λίστα περιέχει τρόπο σύνδεσης με τον πρώτο σύνδεσμο της λίστας.

Στοίβα

Littel words:

Stack is an abstract data type with a bounded(predefined) capacity. It is a simple data structure that allows adding and removing elements in a particular order. Every time an element is added, it goes on the top of the stack and the only element that can be removed is the element that is at the top of the stack, just like a pile of objects.

Basic features of Stack:
  1. 1.Stack is an ordered list of similar data type.
  2. 2.push() function is used to insert new elements into the Stack and pop() function is used to remove an element from the stack. Both insertion and removal are allowed at only one end of Stack called Top.

Ουρά

Little words:

Queue is also an abstract data type or a linear data structure, just like stack data structure, in which the first element is inserted from one end called the REAR(also called tail), and the removal of existing element takes place from the other end called as FRONT(also called head). This makes queue as FIFO(First in First Out) data structure, which means that element inserted first will be removed first.

Basic features of Queue:
  1. 1.Like stack, queue is also an ordered list of elements of similar data types.Queue is a FIFO( First in First Out ) structure.
  2. 2.peek()function is oftenly used to return the value of first element without dequeuing it.

Συνδεδεμένη λίστα

Littel words:

Linked List is a very commonly used linear data structure which consists of group of nodes in a sequence. Each node holds its own data and the address of the next node hence forming a chain like structure. Linked Lists are used to create trees and graphs.

The concept of Linked List:
  1. 1.Link − Each link of a linked list can store a data called an element.
  2. 2.Next − Each link of a linked list contains a link to the next link called Next.
  3. 3.LinkedList − A Linked List contains the connection link to the first link called First.

Δυαδικό δέντρο

Littel words:

A binary tree is a hierarchical data structure in which each node has at most two children generally referred as left child and right child.

Each node contains three components:
  1. 1.Pointer to left subtree
  2. 2.Pointer toright subtree
  3. 3.Data elemennt

Binary Tree: Common Terminologies

  • Root: Topmost node in a tree.
  • Parent: Every node (excluding a root) in a tree is connected by a directed edge from exactly one other node. This node is called a parent.
  • Child: A node directly connected to another node when moving away from the root
